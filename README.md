# Wikipedia Search Application

A simple Wikipedia Search App built using HTML, CSS, Bootstrap, and JavaScript.  
It allows users to search Wikipedia articles and displays results dynamically using an API.

## Features

- Search Wikipedia articles
- Press **Enter** to search
- Shows loading spinner while fetching data
- Displays article title, URL, and description
- Opens results in a new tab

## Technologies Used

- HTML
- CSS
- Bootstrap
- JavaScript
- Fetch API
